## Required pre-installed packages Ubuntu 18.04

* **git**
    * sudo apt-get git
* **agrep**
    * sudo apt-get agrep
* **cmake** 
    * sudo apt-get cmake       
* **gcc**
    * sudo apt-get gcc
* **g++**
    * sudo apt-get g++
* **unzip**
    * sudo apt-get unzip
* **sed**
    * sudo apt-get sed
* **python**
    * sudo apt-get install software-properties-common
    * sudo add-apt-repository ppa:deadsnakes/ppa
    * sudo apt-get update
    * sudo apt-get install python3.7
* **libtinfo**
    * sudo apt-get libtinfo-dev
